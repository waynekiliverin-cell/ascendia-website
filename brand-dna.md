# Brand DNA — ASCENDIA 的品牌基因

> 所有场景、所有页面共享的底层规范。无论哪种设计风格，都不能违背此文件的核心约束。  
> ASCENDIA 的视觉不是传统留学中介，也不是 AI 生成的通用 Landing Page，而是一个专业、可信、高端、有陪伴感的 boutique career studio。

---

## 🎨 品牌固定三色

ASCENDIA 的主视觉来自 logo 的深蓝、金色与银白气质，整体传递「专业可信、国际化、高端陪跑、结果导向」的品牌印象。

| 色名 | 色值 | 对应 | 用途 |
|------|------|------|------|
| 深海军蓝 | `#0B1324` | 主背景 / 职业信任感 | 主色调、Hero 背景、导航栏、深色卡片、品牌主视觉 |
| 品牌金 | `#D6B85F` | 成长 / 机会 / 高端感 | CTA、重点数字、icon、标签、连接线、标题强调 |
| 银白灰 | `#E3E7EA` | 理性 / 清晰 / 专业感 | 正文、分割线、辅助信息、浅色文字 |

三色比例原则：深蓝 60% · 金色 25% · 银白 15%。  
金色永远是强调色，不做大面积铺底；深蓝负责建立品牌质感，银白负责保证信息清晰度。

## 🧭 品牌定位规范

ASCENDIA 是面向留学生的高端职业规划与求职陪跑工作室，核心不是制造焦虑，而是通过结构化方法帮助学生看清方向、补齐表达、提升命中率。

### 品牌关键词

- 留学生职业规划
- 高端求职工作室
- 外企 / 金融 / 咨询 / 商业岗位
- 2v1 导师矩阵
- 简历与 LinkedIn 重塑
- Storybank 搭建
- Mock Interview
- Networking 实战
- Redefine Your Limits
- Professional but human

### 品牌不是

- 传统留学中介
- 网红式求职号
- 低价模板化简历修改
- 只会堆砌术语的求职机构
- 贩卖焦虑的培训班
- AI 生成的通用职业网站

## 🔤 字体基因

### 核心原则

- **标题要有高级感**：适合使用衬线字体，制造精品咨询感
- **正文要极度清晰**：信息密度高，但不能压迫
- **英文可作为品牌装饰**：用于 slogan、标签、模块名、数据旁注
- **数字要有冲击力**：成功案例、offer rate、服务成果可用大号数字突出
- **避免过度互联网感**：不要使用太常见的 SaaS 模板字体组合

### 推荐字体池

| 场景 | 推荐 | 备注 |
|------|------|------|
| 英文品牌标题 | `Fraunces` / `Playfair Display` / `Cormorant Garamond` | 高端、精英、咨询感 |
| 中文标题 | `Noto Serif SC` / `Source Han Serif SC` / `霞鹜文楷` | 有专业感，不幼稚 |
| 中文正文 | `Noto Sans SC` / `Source Han Sans SC` | 清晰、稳定、适合长文案 |
| 英文正文 | `Manrope` / `Avenir` / `Helvetica Neue` | 干净、现代、外企感 |
| 数据与标签 | `IBM Plex Sans` / `Fira Code` | 用于数据、进度、步骤、系统感模块 |
| 注释 / mentor 批注 | `Caveat` / `霞鹜文楷` | 少量使用，制造陪伴感和人工感 |

### 字号系统

- Hero 大标题：`clamp(3rem, 7vw, 6rem)`
- Section 标题：`clamp(1.8rem, 4vw, 3rem)`
- 卡片标题：`1.15rem ~ 1.5rem`
- 正文：`16px ~ 18px`
- 辅助文字：`0.78rem ~ 0.9rem`
- 大装饰数字：`clamp(3.5rem, 8vw, 7.5rem)` + `opacity: 0.12~0.18`
- CTA 按钮文字：`0.95rem ~ 1.05rem`，字重要稍重

## ✨ 气质关键词

设计出来的东西应该让人觉得：

- **高端但不高冷** — 有质感，但不让学生有距离感
- **专业但不压迫** — 结构清晰，但不贩卖焦虑
- **外企感但不模板化** — 像成熟职业咨询品牌，不像 SaaS 官网
- **结果导向但有人味** — 强调 offer、命中率和方法论，但保留 mentor 陪伴感
- **精英感但不装** — 讲能力、讲路径、讲真实经验，不讲空泛成功学
- **一看就是 ASCENDIA** — 深蓝金色、高端求职、结构化陪跑、Redefine Your Limits

## 🧩 视觉系统

## 🖼️ Logo 使用规范

ASCENDIA logo 是品牌识别核心，页面中应保持清晰、克制、高级。

### 使用规则

- logo 优先放在深蓝背景上
- logo 周围必须有足够留白
- 不允许拉伸、压缩、旋转 logo
- 不允许给 logo 添加 neon、投影、描边或玻璃效果
- 不建议把 logo 放进复杂图片或高噪音背景
- 小尺寸场景中只保留清晰版本，不做过度装饰

### 推荐位置

- 首页 Hero 左上角或中心偏上
- 导航栏左侧
- Footer 品牌区
- 预约诊断页顶部
- 成功案例页角标
- 下载版 PDF / pitch deck 封面

---

## 🧱 页面布局原则

ASCENDIA 的页面不能所有 section 都居中。整体应采用「左对齐 + 分栏 + 数据卡片 + 流程结构」的组合，体现职业咨询机构的专业判断和信息组织能力。

### 推荐布局

- Hero：左右分栏，左侧品牌主张，右侧 logo / 数据 / 服务卡片
- 数据区：大号数字横向排列，突出成功案例和 offer 表现
- 服务区：三到四个核心服务模块，采用错落卡片而非普通网格
- 方法论区：用 timeline / stepper 展示求职陪跑路径
- 案例区：用真实感强的 case card，不用模板化评价墙
- CTA 区：深蓝底 + 金色按钮 + 简短有力文案
- Footer：克制、清晰、可信任，不做复杂装饰

### 间距原则

- Section 之间：`clamp(88px, 12vh, 160px)`
- 内容块之间：`clamp(40px, 6vw, 96px)`
- 卡片内 padding：`clamp(28px, 3vw, 44px)`
- 元素间 gap：`clamp(24px, 3vw, 48px)`
- 内容最大宽度：`max-width: 1280px`
- 页面左右边距：桌面端 `48px~72px`，移动端 `20px~24px`

## 📝 文案语气

ASCENDIA 的文案应该像一个有经验、有判断力、真诚负责的 mentor，而不是销售。

### 推荐语气

- 清晰
- 克制
- 可信
- 有陪伴感
- 结果导向
- 不制造焦虑
- 不夸大承诺
- 有真实职场经验感

### 推荐表达

- “我们帮你把复杂的求职路径拆成清晰的行动计划。”
- “不是制造焦虑，而是陪你看清方向、补齐能力、提高命中率。”
- “从简历、故事库、模拟面试到 networking，我们关注的不只是申请结果，而是你是否真正具备进入目标行业的表达能力和竞争力。”
- “Redefine Your Limits，不是喊口号，而是通过结构化陪跑，让你的经历被更准确地看见。”
- “让每一段经历，都能被目标岗位看懂。”
- “把模糊的职业焦虑，变成可执行的求职路径。”

### 禁止表达

- “保 offer”
- “逆天改命”
- “全网最强”
- “独家内幕”
- “不报就晚了”
- “普通留学生也能轻松上岸”
- “100% 拿 offer”
- “碾压同龄人”
- “割韭菜式焦虑话术”

## 📊 数据展示规范

ASCENDIA 可以使用关键数据建立信任，但必须克制、可信、不过度营销。

### 推荐数据展示

- `200+` 成功辅导案例
- `92%` 上岸率
- `180%` 人均 offer 率
- `2v1` 导师矩阵
- `3h` 工作时间响应
- `3 days` 首次 call 后交付周期

### 数据视觉规则

- 数字可以大，但解释要小
- 数字用金色或银白，不用红色
- 不使用夸张闪烁动效
- 不用廉价战报风
- 每个数据旁边必须有清楚语境
- 数据区整体要像 consulting report，不像广告海报

## 🧑‍🏫 Mentor 感设计

ASCENDIA 的差异化不是“卖课”，而是“高质量 mentor 陪跑”。因此页面可以适度加入 mentor 感细节。

### 可使用元素

- 批注式小标签
- 简历修改前后对比
- 面试问题拆解卡
- Storybank 结构示意
- Networking message 示例
- 进度追踪小组件
- Next Step 模块
- “导师建议”提示框

### 注意

这些元素要像真实工作流，不要像装饰。  
所有 mentor 感设计都应服务于“可信任”和“可执行”，不要变成可爱贴纸或小红书风。

---

## ✨ 动效风格

动效应克制、稳定、有质感。ASCENDIA 的动效不是为了炫技，而是为了帮助用户理解信息层级。

### 推荐动效

- subtle fade in
- slow reveal
- card hover lift
- underline slide
- number count-up
- soft glow on CTA
- timeline step reveal
- progress bar fill

### 禁止动效

- bounce / elastic
- 无限循环漂浮
- 夸张 parallax
- 快速闪烁
- neon 发光
- 大面积粒子动画
- animate width / height 导致页面抖动
- 任何让页面像 AI 模板或 Web3 官网的动效

### 无障碍要求

必须尊重 `prefers-reduced-motion`。  
减少动效时，页面仍然要完整可读，不隐藏核心信息。

## 🚫 通用禁忌清单

| 类型 | 禁止 |
|------|------|
| 配色 | 蓝紫渐变、neon、cyan、大面积纯黑白、廉价金色、过度冷灰蓝 SaaS 感 |
| 字体 | Inter / Roboto / Arial 作为主字体、过度科技感 monospace、过于可爱的圆体 |
| 布局 | 所有 section 居中、千篇一律卡片网格、cards 嵌套 cards、模板化 Landing Page |
| 动效 | bounce / elastic、无限循环动画、过度视差、闪烁光效 |
| 装饰 | glassmorphism、渐变文字、AI 光效、无意义 3D 图、stock photo 风人物图 |
| 文案 | 贩卖焦虑、夸大承诺、保 offer、全网最强、逆袭话术 |
| 整体 | 看起来像 AI 生成的通用模板、像传统留学中介、像低价简历修改工作室 |
| 默认样式 | HTML 默认 blockquote、默认 border-left 引用块、无样式 ul/ol、默认 table |
| 图片 | AI 生成的职场 stock photo、过度滤镜、假笑商务人物图 |
| 边框 | 不要大面积厚重边框；金色边线应细、克制、有节奏 |
| 排版 | 行间距/字间距必须肉眼检查，不允许出现过松或过紧的异常节奏 |

### 自检问题

做完设计后问自己：

1. 这个页面看起来像高端 career studio，还是像普通留学中介？
2. 这个页面有没有 ASCENDIA 的深蓝金色识别感？
3. 这个页面有没有真实 mentor 陪跑的温度？
4. 这个页面有没有过度营销或贩卖焦虑？
5. 这个页面截图发出去，会不会被人觉得“又是 AI 模板”？
6. 用户能不能在 10 秒内理解：ASCENDIA 是谁、帮我做什么、为什么可信？

- Section 之间：`clamp(88px, 12vh, 160px)`

- 卡片内 padding：`clamp(28px, 3vw, 44px)`

## 📱 响应式通用规则

- 断点：`1024px` 三栏变两栏
- 断点：`900px` 两栏变单栏
- 断点：`600px` 字号与按钮微缩
- 移动端是“重新排列”，不是简单缩小
- 移动端不隐藏核心内容
- CTA 在移动端应更靠前
- 数据卡片在移动端可以纵向堆叠
- 导航栏移动端应简洁，不做复杂动画菜单

## 🔍 细节规范

### 文本选中

```css
::selection {
  background: #D6B85F;
  color: #0B1324;
}
```

### 链接悬停

链接 hover 使用金色下划线或金色底部滑线，不只依赖变色。

```css
a {
  color: #E3E7EA;
  text-decoration: none;
  background-image: linear-gradient(#D6B85F, #D6B85F);
  background-size: 0% 2px;
  background-repeat: no-repeat;
  background-position: 0 100%;
  transition: background-size 0.25s ease;
}

a:hover {
  background-size: 100% 2px;
}
```

### CTA 按钮

CTA 使用金色渐变或纯金色，文字用深蓝，不使用红色大按钮。

```css
.btn-primary {
  background: linear-gradient(135deg, #D6B85F 0%, #F1D884 100%);
  color: #0B1324;
  border-radius: 999px;
  font-weight: 700;
  box-shadow: 0 12px 30px rgba(214, 184, 95, 0.24);
}
```

### 卡片

卡片应有轻微边框和层次，不使用玻璃拟态。

```css
.card {
  background: #111B2E;
  border: 1px solid #263247;
  border-radius: 24px;
  box-shadow: 0 20px 60px rgba(0, 0, 0, 0.24);
}
```

ASCENDIA 官网至少应包含以下模块：

1. Hero 品牌主张  
   Redefine Your Limits / 打破职场天花板

2. 核心数据区

7. 品牌承诺  
   交付周期、响应机制、next step、进度追踪

8. 免费 1v1 诊断 CTA  
   引导用户提交学校、年级、目标方向

## 🧠 最终品牌判断标准

ASCENDIA 的设计必须同时满足以下三点：

1. **第一眼有质感**  
   用户觉得这是一个专业、高端、值得信任的 career studio。

2. **第二眼有方法**  
   用户能看见清晰的服务路径、导师体系和求职方法论。

3. **第三眼有温度**  
   用户感到这不是一个制造焦虑的机构，而是真正有人陪他一起拆解问题、推进结果。
